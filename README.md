# piano-cartesiano-java

Programma in Java che disegna la funzione inserita:

![alt text](https://github.com/ErliPan/piano-cartesiano-java/blob/master/screenshot.png)


Esempio:

f(x) = cos(X)+4

f(x) = sin(Cos(x/2)) +5

f(x) = x^3+ 5

f(x) = (x +4) ^ 4

f(x) = sin(x^2)

f(x) = 20 *sin(x^3/10)


Nota: Gli spazi sono ignorati
